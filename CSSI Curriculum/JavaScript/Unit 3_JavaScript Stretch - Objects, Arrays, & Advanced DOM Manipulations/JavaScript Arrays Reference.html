CONTEXT

Functions can take various data types and data structures as inputs, and they can produce them as outputs. Incorrectly formatted data structures will generally cause a function to produce an error. Databases can aggregate many different data types and data structures, and that data can then be used in various scripting languages to display data to a user, to build HTML or CSS code, or to generate new data.

Arrays In Action

Arrays are collections of elements separated by commas, but arrays maintain a sequential order whereas the key-value pairs of an object do not. The elements in an array can be of any type, including other arrays or even objects. Arrays are denoted by square brackets [].

1
2
3

const emptyArray = [];

const fruits = ['banana', 'apple', 'strawberry'];

Arrays can be declared over many lines:

1
2
3
4
5

const fruits = [
  'banana',
  'apple',
  'strawberry'
];

You can get the length of an array with .length property (no parentheses):

1

console.log(fruits.length); // prints 3

Accessing Elements in Arrays

Elements in an array are accessed by an index number. An index number is that element’s position when counted from zero (zero-based indexing). In the following example:

1

const fruits = ['banana', 'apple', 'strawberry'];

banana has index 0, apple has index 1, strawberry has index 2

Access an element by using its index in square brackets:

1
2
3

console.log(fruits[0]); // prints 'banana'
console.log(fruits[2]); // prints 'strawberry'
console.log(fruits[4]); // prints undefined

Adding and Setting Elements in Arrays

Add elements to the end of an array by using the .push() function:

1
2
3

const fruits = ['banana', 'apple', 'strawberry'];
fruits.push('orange');
console.log(fruits); // prints ['banana', 'apple', 'strawberry', 'orange']

Change elements in an array by assigning a value to the array’s index directly:

1
2
3

const fruits = ['banana', 'apple', 'strawberry'];
fruits[1] = 'grape';
console.log(fruits); // prints ['banana', 'grape', 'strawberry']

There are many other methods for manipulating arrays, including slice(), splice(), pop(), concat(), shift(), unshift(), and more.

Looping over Arrays

One way to loop over an array is to use the native JavaScript forEach() function on an array. This function accepts a single argument, a function, that will execute for each element in the array.

1
2
3

fruits.forEach((fruit) => {
  console.log(fruit);
});

Prints:

1
2
3

banana
apple
strawberry

If you need the index of each element you’ll have to include that as the second argument in the anonymous function, shown below as i:

1
2
3

fruits.forEach((fruit, i) => {
  console.log(i + ':' + fruit);
});

Prints:

1
2
3

0:banana
1:apple
2:strawberry

Note that you will need the index if you plan on changing any elements of the array (since that’s how we access a specific element).

You can also loop over an array by writing a for loop starting at 0 and ending at the length of the array. Here, i once again represents the current index:

1
2
3
4

for (let i = 0; i < fruits.length; i++) {
  const fruit = fruits[i];
  console.log(i + ':' + fruit);
}

Prints:

1
2
3

0:banana
1:apple
2:strawberry

Tips & Tricks

  • There are many resources on the Web for learning how to manipulate arrays:
    • JavaScript Arrays
    • JavaScript Array Methods
  • Use a linter (e.g. JSONLint) to ensure your arrays are properly formatted and that you have no superfluous commas or quotes.

Best Practices Summary

  • Arrays are denoted by square brackets, [], and elements in an array are separated by commas, ["a", 2, "C", true].
  • Get the length of an array using the .length property.
  • Elements in an array are accessed by their index number, beginning at 0. The index is put in square brackets after the name of the array: arrayName[0].
  • Add elements to the end of an array using .push().
  • Loop through the elements in an array using a for(){ ... } loop or the .forEach() function.
