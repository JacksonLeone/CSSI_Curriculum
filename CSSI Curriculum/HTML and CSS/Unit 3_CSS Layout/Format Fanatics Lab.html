CONTEXT

CSS is the language of style and design on the World Wide Web, and CSS Flexbox and CSS Grid are subsets of CSS properties that will enable you to quickly create layouts that readily adapt to various sized screens.

CSS Grid

Initialize CSS Grid

To use CSS Grid, set the display property of the container .wrapper to grid or inline-grid. All direct children of the parent now become grid items and the auto-placement algorithm lays them out, one in each grid cell.

By default, CSS Grid creates 1 column.

1
2
3

.wrapper {
  display: grid;
}

Grid Column & Rows

Grid Column

You can define how many columns you want and the width of each column with the property grid-template-columns.

1
2
3
4

.wrapper {
  display: grid;
  grid-template-columns: 100px 100px 100px;
}

We can also use the value auto to make the width of a column take up as much room as is remaining on our page.

1
2
3
4

.wrapper {
  display: grid;
  grid-template-columns: 100px 100px auto;
}

Grid Rows

You can define however many rows you want and the height of each row with the property grid-template-rows.

1
2
3
4
5

.wrapper {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 200px 300px;
}

Fraction Units

Another way of setting the value for your columns and rows is to use fraction units. You can use the value fr to denote the value of your column or rows as a portion of all remaining space. A column 2fr wide is always twice as wide as a column 1fr wide.

1
2
3
4

.wrapper {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
}


1
2
3
4
5

.wrapper {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
  grid-template-rows: 1fr 3fr;
}

Repeat

Declaring six equally sized rows can seem tedious: 1fr 1fr 1fr 1fr 1fr 1fr. Instead, we can use repeat to help us define our template. Two parameters are required: the quantity (of columns or rows) and the space that each should take up (height for row, width for column).

1
2
3
4
5

.wrapper {
  display: grid;
  grid-template-columns: repeat(6, 1fr);
  grid-template-rows: 100px;
}

Grid Gap

You can add space in between the elements by adding a value to the property grid-gap.

1
2
3
4
5
6

.wrapper {
  display: grid;
  grid-gap: 10px;
  grid-template-columns: 2fr 1fr 1fr;
  grid-template-rows: 100px 300px;
}

TIPS & TRICKS

  • Resources on the web like Grid by Example, a website dedicated to CSS Grid Layout examples, can help you quickly get started.
  • Try drawing out the layout of your site before you start coding it. This can help you plan ahead and troubleshoot how columns will flow on larger vs. smaller screens.

BEST PRACTICES SUMMARY

  • To begin using CSS Grid, add display: grid; to a container element.
    • Use the grid-template-columns property to define columns; use the grid-template-rows property to define rows.
    • Dimensions of columns and rows can be in pixels (200px), fractional widths (2fr), or auto-filling (auto).
    • Use repeat() to write concise values.
    • Separate columns and rows using the grid-gap property.
