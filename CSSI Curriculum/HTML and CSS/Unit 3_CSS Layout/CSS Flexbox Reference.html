CONTEXT

CSS is the language of style and design on the World Wide Web, and CSS Flexbox and CSS Grid are subsets of CSS properties that will enable you to quickly create layouts that readily adapt to various sized screens.

Css Flexbox In Action

Parent Flex Properties

All the properties included in this section are meant to be set on the parent element.

Initializing Flexbox

Use display: flex; to give the items inside a parent element some amount of flexibility in terms of their size and positioning.

Setting the Flex Direction

The four values for flex-direction you are likely to see are row, column, row-reverse, and column-reverse.

  • flex-direction: row; is the default value; child elements will be displayed horizontally in a row.
  • flex-direction: column; will display child elements vertically in a column.
  • flex-direction: row-reverse; will display child elements in a row but in reverse order.
  • flex-direction: column-reverse; will display child elements in a column but in reverse order.

Aligning Items

The four values for align-items you are most likely to see are stretch, flex-start, flex-end, and center. You may also see baseline.

  • align-items: stretch; is the default behavior and child elements are stretched to fit in the parent element.
  • align-items: flex-start; will position child elements starting at the beginning of the parent element.
  • align-items: flex-end; will position child elements at the end of the parent element.
  • align-items: center; will position child elements at the center of the parent element.
  • align-items: baseline; will position child elements at the baseline of the parent element.

Justifying Content

The justify-content property is arguably one of the most commonly used and the most powerful. The values you’ll see most often associated to it can be broken up into two categories:

Values that cluster content together:

  • justify-content: flex-start; is the default value and will position child elements at the beginning of the parent element.
  • justify-content: flex-end; will position child elements at the end of the parent element.
  • justify-content: center; will position child elements at the center of the parent element.

Values that space content apart:

  • justify-content: space-between; will position child elements with space between each child element.
  • justify-content: space-around; will position child elements with space between and around each child element.
  • justify-content: space-evenly; will position child elements with equal amounts of space between and around each child element.

Combining Properties

Each of these properties can is useful on its own, but some of the most striking visual results are achieved by combining two or more properties. Experiment with combinations and examine the results.

Child Flex Properties

Assigning properties to child elements tends to be most useful when you want one of the child elements to behave differently from its siblings.

Expanding a Child Element

The flex-grow property defines how much a child element will expand relative to other child elements within the same parent element.

  • flex-grow: 2; will make a child element twice as wide as other unmodified child elements.

Tips & Tricks

  • Resources on the web like A Guide to Flexbox can help you quickly get started.
  • Try drawing out the layout of your site before you start coding it. This can help you plan ahead and troubleshoot how columns will flow on larger vs. smaller screens.

Best Practices Summary

  • To begin using CSS Flexbox, add display: flex; to a container element.
    • Use the flex-direction property to change the direction (row vs. column) and order in which child elements are shown.
    • Use the align-items property to change how child elements are arranged in a parent element.
    • Use the justify-content property to change how child elements are distributed within a parent element.
  • Use CSS Flexbox properties like flex-grow on a child element when you want that element to behave differently than its siblings.
