CSS Quiz 2: Open Response

1. CSS Selectors

What is the correct HTML syntax for assigning an id of main-title to an h1 tag?

Answer

<h1 id="main-title"></h1>

2. CSS Selectors

Given the following code, how would you select all of the paragraphs and give them a margin-top of 5%?

1
2
3
4
5
6

<div>
  <p>The first thing that I do when I wake up in the morning...</p>
  <p>is to get out of bed and stretch.</p>
  <h2>My success plan</h2>

4. Margins & Padding

Given the following HTML, write the CSS code to give the text a padding of 10px on the right and 10px on the bottom. In addition, the element should have a margin of 5% on the top.

1

<div id="hippo">The purple hippopotamus</div>

Answer

1
2
3
4
5

#hippo {
  padding-right: 10px;
  padding-bottom: 10px;
  margin-top: 5%;
}

Or a variation on that such as:

1
2
3
4

#hippo {
  padding: 0 10px 10px 0;
  margin: 5% 0 0 0;
}

5. OPTIONAL: CSS Grid

Based on the HTML code below, use CSS Grid to make your web page display these divs in two columns. The column on the left should be 100px and the column on the right should automatically take up the rest of the space. Specify that the row heights should be 20px for the first and third row, but the second row should automatically adjust to the height of the content.

1
2
3
4
5
6
7
8

<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</div>

Answer

1
2
3
4
5

.container {
  display: grid;
  grid-template-columns: 100px auto;
  grid-template-rows: 20px auto 20px;
}
